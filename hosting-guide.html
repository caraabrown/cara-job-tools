If your host is Squarespace / Wix / Webflow

+

These platforms usually do not allow raw multi-file uploads in the same way. Use one of these options:

+
    +
  • Recreate each page as native platform pages and copy content from these files.
    • +
  • Or host this mini-site separately on Netlify/Vercel/GitHub Pages and link from your main domain (for example: tools.caraabrown.com).
    • +
+
+
+ +
+
+

If your host uses cPanel or FTP

+
    +
  1. Open File Manager or your FTP client.
    2. +
  2. Go to public_html (or your domain root folder).
    3. +
  3. Upload all files and folders from this project root.
    4. +
  4. Visit https://caraabrown.com/index.html and test each page link.
    5. +
